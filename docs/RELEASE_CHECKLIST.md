# Release Checklist

Use this checklist before publishing a Lyra build.

## Functional checks

- Browse local drives and common user folders.
- Create, rename, copy, move, and delete test files.
- Verify file previews still open with the expected app.
- Confirm protected folders show a useful access message.

## Package checks

- Build in release configuration.
- Run the packaged app from a clean directory.
- Confirm version metadata and release notes match the build.
